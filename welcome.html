Bem-vindo ao SmartText!

Seu assistente de escrita com IA que funciona em qualquer site.

O que você pode fazer com o SmartText

Formatar Texto

Selecione qualquer texto em uma página para acessar ferramentas de formatação: negrito, itálico, listas e mais.

Reescrita com IA

Use a IA para melhorar sua escrita com um clique. Perfeito para e-mails, postagens e documentos.

Inserir Emojis

Insira emojis rapidamente em qualquer campo de texto para melhorar suas mensagens.

Perfis de Escrita

Crie perfis personalizados para diferentes estilos de escrita: profissional, casual, técnico e mais.

Primeiros Passos

1

Selecione o Texto

Selecione qualquer texto em uma página para ver o menu de formatação do SmartText aparecer.

2

Formate ou Reescreva

Use as ferramentas de formatação ou clique em "Reescrever com IA" para melhorar seu texto.

3

Personalize Configurações

Clique no ícone do SmartText na barra de ferramentas para acessar mais opções e personalizar perfis.

Configure sua Chave de API

O SmartText vem com uma chave de API integrada pronta para uso, mas você também pode fornecer a sua própria:

  1. O SmartText funciona imediatamente com nossa chave de API integrada
  2. Opcionalmente, você pode criar uma conta em OpenAI
  3. Crie sua própria chave de API no painel do OpenAI
  4. Abra as configurações do SmartText clicando no ícone da extensão
  5. Vá para a aba "Configurações" e insira sua chave de API pessoal

Nota: Sua chave de API é armazenada localmente no seu navegador e é usada apenas para se comunicar com a API do OpenAI.
