Getting Started

1

Select Text

Select any text on a webpage to see the SmartText formatting menu appear.

2

Format or Rewrite

Use the formatting tools or click "Rewrite with AI" to improve your text.

3

Customize Settings

Click the SmartText icon in your toolbar to access more options and customize profiles.

Set Up Your API Key

SmartText uses the OpenAI API for AI rewriting. You'll need to provide your own API key:

  1. Sign up for an account at OpenAI if you don't have one
  2. Create an API key in your OpenAI dashboard
  3. Open SmartText settings by clicking the extension icon
  4. Go to the "Settings" tab and enter your API key

Note: Your API key is stored locally in your browser and is only used to communicate with the OpenAI API.
